Speech AI Suite

Professional multi-task speech analysis platform powered by state-of-the-art deep learning models

Project Overview

Speech AI Suite is an advanced audio classification system that leverages cutting-edge self-supervised learning models to perform four critical speech analysis tasks: emotion recognition, gender classification, intent detection, and speaker identification. Built with a clean, modular architecture following modern software engineering principles.

Emotion Classification

Model: HuBERT-large + SVM Classifier

Accuracy: 79.14% (5-fold cross-validation)

Dataset: CREMA-D

Classes: Happy, Sad, Angry, Neutral, Fear, Disgust

Architecture: facebook/hubert-large-ll60k (1024-dim embeddings) with SVM classifier

Gender Classification

Model: WavLM-base-plus + Logistic Regression

Feature Extraction: microsoft/wavlm-base-plus

Classes: Male, Female

Dimensionality Reduction: PCA (200 components)

Use Case: Speaker demographics analysis for personalized services

Intent Classification

Model: WavLM-base-plus + SVM Classifier

Dataset: SLURP (Spoken Language Understanding)

Classes: 20 intent categories (commands, queries, etc.)

Developer: Sahasra Ganji

Use Case: Voice assistant command understanding for smart devices

Speaker Identification

Model: XLSR-53 + Custom Pooling

Feature Extraction: facebook/wav2vec2-large-xlsr-53

Pooling Strategy: Mean + Std concatenation (2048-dim)

Dimensionality Reduction: PCA (200 components)

Use Case: Biometric authentication and speaker diarization

Technical Architecture

Backend Stack
  • Framework: Flask (Python Web Framework)
  • ML Libraries: PyTorch, Transformers, scikit-learn
  • Audio Processing: librosa, soundfile, pydub
  • Model Storage: Git LFS (Large File Storage)
Frontend Stack
  • UI Framework: Bootstrap 5
  • Design: Glassmorphism with gradient backgrounds
  • Icons: Bootstrap Icons
  • Audio Visualization: Custom CSS animations
